# Input Monitor Utility (Ethical Use Only)

A simple, educational keylogger for monitoring keyboard input on your own machine.  
**WARNING:** Only use on systems you own or have explicit permission to monitor.  
Unauthorized use is illegal.

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python -m input_monitor
```
Press `ESC` to stop logging. Logs saved to `keylog.txt`.

## Testing
```bash
python -m pytest tests/
```
